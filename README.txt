Il gioco, per poter funzionare, necessita di avere python e pygame installati.

LINUX:

Installazione:
Da terminale lanciare i comandi:
-'sudo apt-get install python', se richiesta, inserire la password dell'utente connesso
-'sudo apt-get install python-pygame'

Gioco:
Da terminale lanciare i comandi:
-'cd #game_directory_path#'
-'python snake.py'


MAC OSX:

Installazione:

Da terminale lanciare i seguenti comandi:
-'cd #game_directory_path#'
-'sh python_setup.sh'

Gioco:

Da terminale lanciare i comandi:
-'cd #game_directory_path#'
-'python3 snake.py'





WINDOWS:

Installazione:

-Installare una versione di python dal sito http://www.python.it/download/
-Installare la corrispondente versione di pygame dal sito http://pygame.org/download.shtml

Gioco:

Fare doppio click sul file snake.py presente nella cartella del gioco


Per ottenere maggiori informazioni e dettagli sull'installazione di pygame visita la pagina http://www.pygame.org/download.shtml
